# 🚀 Fullstack Authentication System

A production-ready authentication system built with **Next.js (Frontend)** and **Node.js + Express + MongoDB (Backend)**.

---

## ✨ Features

- 🔐 JWT Authentication (httpOnly cookies)
- 📱 Multi-device login tracking
- 🖥️ Device management (view active devices)
- 🚫 Admin force logout (single device / all devices)
- 🛡️ Protected routes (Next.js middleware)
- 🍪 Secure cookie-based authentication
- ⚡ Session management with database
- 🔄 Auto logout when session expires

---

## 🏗️ Tech Stack

### Frontend
- Next.js (App Router)
- TypeScript
- Axios

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT
- Cookie Parser
